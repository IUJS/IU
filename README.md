# IU JS

IUJS is a modern client-side JavaScript framework for building SPAs (Single Page Applications).

> Made with ♥️ by Iranian.

### Development

#### Build flow

```
npm run build
```

#### Check flow

```
npm run check 
```

#### lint-staged

```
npm run test
```

#### Lints

```
npm run lint
```

#### Fix lint errors

```
npm run lint:fix
```
