# IU JS

IUJS is a modern client-side JavaScript framework for building SPAs (Single Page Applications).

> Made with ♥️ by Iranian.

### Development

#### Build flow

```
npm run build
```

#### Check flow

```
npm run check
```

#### Flow lint

```
npm run lint
```

#### lint-staged

```
npm run test
```

#### Check es-lints

```
npm run eslint
```

#### Fix es-lint errors

```
npm run eslint:fix
```
